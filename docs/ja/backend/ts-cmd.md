# Typescriptコマンド

#### プロジェクトディレクトリの作成、初期化
```bash
npm init -y
```

#### Fastifyのインストール
```bash
npm install fastify
``` 

#### tRPCのインストール
```bash
npm install @trpc/server @trpc/client
```

#### 起動
```bash
npx expo start
```

#### ts-nodeのインストール
```bash
npm install -g ts-node
```

#### lintのインストール
```bash
npx eslint .
```
