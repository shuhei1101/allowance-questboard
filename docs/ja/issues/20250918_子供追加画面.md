# 家族招待コード発行画面

# 初期表示
## 現在家族の招待コードを取得する
  - 引数: 
    - jwt(jwtからuseridを取得し、useridに紐づく親ユーザの家族idを取得)
  - 戻り値: 
    - 招待コード
    - 期限
- 戻り値を画面の状態に登録する

# 画面項目

## トップバー
### 戻るボタン
- expo標準の戻るボタン

### タイトル
- `家族招待コード発行`

## コンテンツエリア

### 招待コードラベル
- コピーボタン付きのインプットボックス
  - これは共通コンポーネントとして定義する
  - 非活性にする
- 発行された招待コードを表示する

### 招待コード発行ボタン
- 状態.期限が切れているor状態.期限がundefinedのとき活性

#### 押下時
- 招待コードを発行する
- 状態を更新する

### 招待コードの期限ラベル
- 状態.招待コード期限を表示
