# 家族コード入力画面
- ポップアップで開くことができるウィジェット
- 入力された家族コードをバックエンドで検証
- 戻り値として家族情報を返却

## 画面項目
### 家族コード入力フィールド
- 家族コード値オブジェクト
  - 必須
  - 文字数チェック

### 確定ボタン
- 入力された家族コードをバックエンドに送って、検証
- 呼び出し元に家族情報を返す
- 画面を閉じる

#### キャンセルボタン
- 画面を閉じる


## バックエンド
### ルーター
- 認証: なし
- 引数: 家族コード
- 戻り値: 家族情報

### サービス
- 検証
