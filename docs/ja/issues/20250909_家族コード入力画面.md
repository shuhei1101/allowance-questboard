# 家族招待コード入力画面
- ポップアップで開くことができるウィジェット
- 入力された家族招待コードをバックエンドで検証
- 戻り値として家族情報を返却

## 画面項目
### 家族コード入力フィールド
- 家族招待コード値オブジェクト
  - 必須
  - 文字数チェック

### 確定ボタン
- 入力された家族招待コードをバックエンドに送って、検証
- 呼び出し元に家族情報を返す
- 画面を閉じる

#### キャンセルボタン
- 画面を閉じる


## バックエンド
### ルーター
- 認証: なし
- 引数: 招待コード
- 戻り値: 家族情報

### サービス
- FamilyInviteRepository.findByInviteCode(引数.招待コード)
- 家族=家族リポジトリ.findid(家族id)
- return 家族

### 対象エンティティ
- FamilyInviteEntity.ts
