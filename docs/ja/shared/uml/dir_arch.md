# 本アプリ「お小遣いクエストボード」のディレクトリ構成

```plaintext
lib/
├── application/ ... アプリケーションサービスクラスやデータ受け渡し用のDTOを管理
│   └── <ドメイン名>/ ... ドメイン毎にフォルダを作成。
│
├── domain/ ... ドメインモデルや値オブジェクト、リポジトリインターフェースなど、ドメインに関するクラスを管理。
│   ├── <ドメイン名>/ ... ドメイン毎にフォルダを作成。
│   └── shared/ ... ドメインモデルで共通して使用するクラスを管理。
│
├── firestore_infrastracture/ ... Firestoreのデータベースにアクセスするためのリポジトリの実装クラスを管理。
│   └── <ドメイン名>/ ... ドメイン毎にフォルダを作成。
│
├── playground/ ... ソースコードを試すためのディレクトリ。lib配下でないとホットリロードが効かないため、一時的に配置。
│ 
├── presentation/ ... プレゼンテーション層のコンポーネントを管理。
│   ├── component/ ... 画面の部品となるコンポーネントを管理。再利用性を高く設計する。
│   ├── page/ ... 画面の土台となるコンポーネント。画面遷移した際に最初に表示される。必ずScaffold（土台）を含む。
│   ├── router/ ... go_routerパッケージを使用した画面遷移を管理。
│   ├── screen/ ... pageよりも細かいが、componentよりも大きいコンポーネントを管理。基本的にScaffold（土台）を含まない。
│   └── theme/ ... テーマカラーなどのテーマ情報を管理。
│
└── util/ ... ユーティリティクラスを管理。共通の変換処理やバリデート処理などを管理。
```
