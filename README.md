本リポジトリはiPhone・Android向けに開発中のアプリ「お小遣いクエストボード」に関するものです。
転職のポートフォリオ用に期間限定で公開しているため、一般公開は予定していません。

# アプリ概要
本アプリは、親が「クエスト」と呼ばれるお手伝いや成長を促すタスクを作成し、子供が自由に選んで挑戦できる仕組みです。
クエストを達成すると、お小遣いや経験値を獲得でき、ゲーム感覚で楽しみながら自発的に成長できます。
